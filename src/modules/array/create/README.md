## create()

### Arguments

`first argument is start of number`
`second argument is last of number`

### Returns 

`iterable object form start to end of infinity? with step 1`

### Examples

```js

  const result = create(1, 5)
  
  for(const el of result) {
    console.log(el)
  }
  
  // 1 2 3 4
```
